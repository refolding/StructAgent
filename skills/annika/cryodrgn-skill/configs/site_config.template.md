# cryodrgn-skill site config (TEMPLATE)

This is the **template/example** of the environment report. The real report is
`configs/site_config.local.md`, generated by `scripts/cryodrgn_env_probe.py` (it
is local/private and `.gitignore`d). Do not hand-edit a generated report; re-run
the probe instead. The probe is read-only — it installs nothing, downloads
nothing, makes no network call, runs no cryoDRGN job, and writes only the report.

Generate it:

```text
# [config-state: absent] — read-only probe; safe to run unprompted (installs/downloads/network/jobs: none)
python3 scripts/cryodrgn_env_probe.py --format markdown --output configs/site_config.local.md
# add --live-help where cryoDRGN is already installed: it captures help-only -h/--version
# text and, on an installed host, all 23 cryodrgn + 25 cryodrgn_utils subcommands
# [VALIDATED: cryoDRGN 4.2.1] (captured 2026-06-06 on a Linux + NVIDIA GPU host; see
# cryodrgn.help.txt, cryodrgn_utils.help.txt)
```
